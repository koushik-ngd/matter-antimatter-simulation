
# Matter-Antimatter Particle Simulation
A real-time particle physics simulation built with Python and Pygame that models matter and antimatter collisions, annihilation events and energy conversion using E = mc2.
What it simulates

Electron and positron collisions
Proton and antiproton collisions
Annihilation events with gamma-ray photon bursts
Elastic scattering between same-type particles
Relativistic kinetic energy using the Lorentz factor
Live tracking of annihilation count and total energy released

How to run
Install dependencies:
pip install pygame
Run the simulation:
python particle_sim.py
Controls

E - Spawn electron/positron pair
P - Spawn proton/antiproton pair
C - Clear all particles
Space - Pause and resume
Left click - Place matter particle
Right click - Place antimatter particle
Hold Shift + click - Place proton type
